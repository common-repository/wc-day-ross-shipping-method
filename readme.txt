== Description ==
This plugin will help you use <a href="https://dayross.com/" target="_blank">Day & Ross</a> Services in the WooCommerce plugin by adding a new shipping method to your shop.
Also, it's completely compatible with Woocommerce and uses the Day & Ross API.

== Available features ==
* Get Rates from Day & Ross API
* Save Shipment Details into The Orders
* Rate Name Adjustment
* Rate Price Adjustment
* Rate Delivery Adjustment
* Product Settings for Special Services
* Product Settings for Additional Services

=== Elite paid features ===
* Submit Pickups to Day & Ross API
* Print Shipping Labels

Please note that this plugin has been made for those who want to submit and manage their shipments from the main service site, which means _the plugin will not send any pickup requests to Day & Ross servers_.
